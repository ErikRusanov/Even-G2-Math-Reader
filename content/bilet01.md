## Машинная арифметика

Множество чисел с плавающей точкой $F$ задаётся основанием $p$, длиной мантиссы $t$ и интервалом порядков $[L,U]$; каждое $x\in F$:

$$
x=\pm\left(\frac{d_1}{p}+\dots+\frac{d_t}{p^t}\right)p^{\alpha},
\quad 0\le d_i\le p-1,\ \ L\le\alpha\le U .
$$

$F$ *нормализовано*, если $d_1\ne0$ для всех $x\ne0$.

**Определение (округление).** Округление с точностью $\eps$ — отображение $\mathrm{fl}\colon\R\to F$ такое, что для любого $y$ с $\mathrm{fl}(y)\ne0$ выполнено $\mathrm{fl}(y)=y\,(1+\eta)$, $|\eta|\le\eps$.

Тогда относительная погрешность $\le\eps$. При традиционном округлении $\eps=\tfrac12 p^{1-t}$, при отбрасывании разрядов $\eps=p^{1-t}$; величина $\eps$ — *машинная точность*. Для `double` (1+11+52 бит): диапазон $\sim10^{\pm308}$, $\eps=2^{-53}\approx10^{-16}$.

## Три требования вычислительной математики

1. **Устойчивость задачи**: малое изменение входных данных не меняет результат значительно.
2. **Численная устойчивость алгоритма**: ошибки округления в промежуточных вычислениях не искажают ответ.
3. **Достаточность ресурсов** (память, быстродействие, ПО) для получения ответа за приемлемое время.

**Пример (неустойчивая задача).** У возмущённой матрицы Уилкинсона $20\times20$ имеем $\det A(\eps)=20!-\eps\cdot20^{19}$. Изменение одного элемента на $\eps=20^{-19}\cdot20!\approx5\cdot10^{-7}$ обращает определитель из $20!\approx2.4\cdot10^{18}$ в нуль и сдвигает $\lambda_{\min}$ с $1$ до $0$: задачи вычисления $\det$ и собственных значений неустойчивы.

**Пример (неустойчивый алгоритм).** Сумму $\sum_{i=1}^{10^3} 1/i^2$ следует накапливать от меньших слагаемых к большим. При сложении сильно различающихся по величине чисел младшие разряды теряются: например при $p=10,\ t=5$ имеем $\mathrm{fl}(100{,}01)+\mathrm{fl}(0{,}0001)=\mathrm{fl}(100{,}01)$ — добавка пропадает. Аналогично теряются значащие цифры при вычитании близких чисел.

**Пример (недопустимо долгий алгоритм).** Метод Крамера для системы $n\times n$ требует $\sim n(n+1)!$ действий (миноры). Для $n=100$ это $\sim10^{162}$ операций — недостижимо ни на каком оборудовании; вдобавок вычисление определителя неустойчиво. Метод неприменим.
